# ⚔️ IRON FALL — Last Soldier Standing

A browser-based top-down shooter with infinite waves, leaderboard, achievements, and more.

## 🚀 Deploy to Vercel

1. Upload this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) → **Add New Project**
3. Import your GitHub repo
4. Click **Deploy** — no configuration needed!

## 🎮 How to Play

- **Move:** W A S D (customizable in Settings)
- **Aim:** Mouse
- **Shoot:** Left Click
- **Exit game:** ESC

## ✨ Features

- ♾️ Infinite waves with scaling difficulty
- 🏆 Top 50 Season Leaderboard
- 🎖️ 12 Achievements
- 💬 10 Unlockable Titles
- 🪖 200 Character & Weapon Skins
- ⚙️ Full Settings (theme, key bindings)
- 🌙 Dark / ☀️ Light mode
